Javascript 연산자(Javascript operators)

1. 산술연산자(Unary operators)

+, -, *, /, %

2. 증감연산자(Increment and Decrement)

age++, age--, ++age, --age

3. 관계연산자(Reational operators)

<,>, <=,>=

4. 동등연산자(Equality operators)

==, !=, ===, !==

== vs === ??

5. 논리연산자(Logical operators)

&&, ||

6. 삼항연산자(Conditional ternary operators)

(condition ? ifTrue : ifFalse)

7. 할당연산자(Assignment operators)

=, +=, -=, *=, /=
